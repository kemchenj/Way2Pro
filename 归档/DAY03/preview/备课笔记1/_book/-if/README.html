2.输出对应星期

3.输出对应季节

  • 接收用户输入的一个整数month代表月份，根据月份输出对应的季节

4.获取两个数最值

  • 键盘输入两个整数，判断大小后输出较大的那个数

5.获取三个数最值

  • 从键盘输入3个整数，判断大小后输出较大的那个数
