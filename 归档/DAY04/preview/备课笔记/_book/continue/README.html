1.continue关键字

  • continue语句的作用是跳过循环体中剩余的语句而继续下一次

  • 使用场合:

    • 循环结构

  • 练习: 把100~200之间的不能被3整除的数输出
    for(int i = 100; i<= 200; i++)
{
  if(i %3 == 0) continue;
  printf("i = %d", i);
}
