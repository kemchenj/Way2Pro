# 结构体数组
##本小节知识点:
1. 【理解】结构体数组的概念
2. 【掌握】结构体数组定义
3. 【掌握】结构体数组初始化

---

##1.结构体数组的概念
- 数组的元素也可以是结构类型的。因此可以构成结构型数组。```结构数组的每一个元素都是具有相同结构类型的下标结构变量。```在实际应用中,经常用结构数组来表示具有相同数据结构的一个群体。如一个班的学生档案,一个车间职工的工资表等。
---


##2.结构体数组定义
- 跟结构体变量一样，结构体数组也有3种定义方式
    + 先定义结构体类型，再定义结构体数组

```
struct Student {
    char *name;
    int age;
};
struct Student stu[5]; //定义1

```

    + 定义结构体类型的同时定义结构体数组

```
struct Student {
    char *name;
    int age;
} stu[5]; //定义2

```

    + 匿名结构体定义结构体结构体数组

```
struct {
    char *name;
    int age;
} stu[5]; //定义3

```
---

##3.结构体数组初始化

- 定义的同时进行初始化
```
struct {
    char *name;
    int age;
} stu[2] = { {”NJ", 27}, {"JJ", 30} };

```
- 先定义,后初始化,整体赋值
```
// 先定义,后初始化,整体赋值
s[1]=(struct stu){23,"xiaoluo"};

// 先定义,后初始化 分开赋值
s[1].age=12;
strcpy(stu[1].name, "xxoo");
```
